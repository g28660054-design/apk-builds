APK Builder
Сборка Android APK прямо с телефона
через GitHub Actions — бесплатно
1
Создай аккаунт GitHub
Если ещё нет — зарегистрируйся на github.com. Это бесплатно. GitHub будет компилировать твой APK в облаке.
2
Создай Personal Access Token
Открой: github.com → Settings → Developer settings → Personal access tokens → Tokens (classic) → Generate new token
Обязательно включи эти разрешения:
repo workflow
Срок действия — можно выбрать No expiration (бессрочный).
Нажми Generate token и сразу скопируй — он показывается только один раз!
3
Вставь токен ниже
Токен начинается с ghp_. Он хранится только в памяти — никуда не отправляется кроме GitHub API.
GitHub Personal Access Token
Вставь токен из GitHub (начинается с ghp_)
